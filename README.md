# GreenKim

### 사용 스택

---

- 안드로이드
    - Retrofit2   Jetpack Compose
- 협업
    - GitHub      Figma

### 주요 기술

---

- '커뮤니티'와 제로웨이스트 활동을 인증하는 '인증'으로 구분된 커뮤니티 환경
- 뱃지와 그에 따른 호칭 획득으로 친환경 라이프 사이클 독려
- 클라이언트와 서버 간의 연결을 통한 데이터 처리와 인증 강화
